# **App Name**: WordPro

## Core Features:

- Offline Vocabulary Storage: Store words and notes locally on the device using SQLite for 100% offline functionality. with PWA
- Dynamic Learning Sessions: Engage in various quiz types, including MCQ (English to Bengali & Bengali to English), spelling tests, fill-in-the-blanks, and verb form tests to keep learning interesting.
- Smart Revision: Prioritize words the user finds difficult ('Hard' and 'Medium' levels) to ensure effective revision before introducing new words using a tool to dynamically adjust questions.
- Detailed Word View: Explore comprehensive details for each word, including meaning, explanation, part of speech, syllables, usage distinction, synonyms, antonyms, and example sentences.
- Add & Manage Words: Easily add new words individually through a form or in bulk via JSON import. Edit or delete existing words at any time.
- Statistics Dashboard: Track learning progress with detailed stats on total words, learned words, accuracy, and error types (spelling vs. meaning).
- Notes: Add simple notes to jot down grammar rules or other language-related thoughts. Support JSON bulk upload. A tool will assist with categorization of notes.

## Style Guidelines:

- Primary color: HSL(210, 68%, 45%) – A strong blue (#2489F7) associated with intellect and focus, suitable for a learning app.
- Background color: HSL(210, 20%, 95%) – Very light blue (#F0F4F8) for a clean, unobtrusive backdrop that keeps the focus on the content.
- Accent color: HSL(180, 70%, 40%) – Teal (#2EAE98) for interactive elements to provide a calm, engaging and complementary feel to the learning experience. Use cautiously to prevent cliche.
- Body and headline font: 'Inter' (sans-serif) for a modern, machined, neutral look that works well for both headlines and body text.
- Use Material Design icons to maintain consistency and clarity across the UI.
- Employ a responsive layout that adapts to different screen sizes using Material 3 design principles.
- Use subtle animations and transitions to provide feedback and enhance the user experience, such as when loading new content or completing a quiz.
